## 📖 Sobre o Projeto

Durante anos, atuei no mercado de **seguros de veículos**, auxiliando clientes na proteção de seus patrimônios, realizando cotações, analisando perfis de risco e construindo relacionamentos através do atendimento consultivo.

Ao iniciar minha transição para a área de tecnologia, decidi unir minha experiência de negócios ao desenvolvimento web, criando um projeto que representa exatamente essa jornada.

O **AutoQuote** é uma landing page para cotação de seguros de veículos desenvolvida com HTML e CSS, simulando uma plataforma moderna de geração de leads para corretoras e consultorias de seguros.

Mais do que um exercício técnico, este projeto demonstra minha capacidade de:

- Entender problemas reais de negócio;
- Transformar necessidades do mercado em soluções digitais;
- Aplicar conceitos de experiência do usuário;
- Evoluir profissionalmente através da tecnologia.

---

## 🎯 Objetivo

Desenvolver uma aplicação web responsiva que simule o processo inicial de captação de clientes para cotação de seguros automotivos.

Este projeto foi criado para consolidar conhecimentos em:

- Estruturação semântica com HTML5;
- Estilização moderna utilizando CSS3;
- Layouts responsivos;
- Boas práticas de desenvolvimento front-end;
- Versionamento de código com Git e GitHub.
